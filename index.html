Use the Sexcoin API to Receive Payments

With the Sexcoin API merchants can easily create a payment URL for purchase of products using Sexcoin. The API tracks the sale with a Sexcoin Address and an invoice number. You don't need an account, just a sexcoin wallet, may we suggest SXCWallet the account is free.


Payments are made to the Input Sexcoin address and then processed to your payment (destination) address.
Use the Callback feature to track incoming invoiced payments to match them to the sale.
